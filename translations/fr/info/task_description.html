Il arrive que vos amis veuillent exprimer leurs sentiments dans un message ou qu'une touche contenant une lettre reste bloquée sur le clavier. Dans les deux cas, nous obtiendrons une "lettre longuement pressée" et la lettre sera alors imprimée plusieurs fois. Vous disposez de deux chaînes de caractères. La première est le message textuel original. La seconde est un message imprimé, qui peut contenir plusieurs (ou éventuellement aucune) lettres longuement pressées. Il peut arriver que le message ait été écrit à la hâte, n'oubliez donc pas de vérifier que toutes les lettres correspondent à celles de l'original. Retournez {% if is_js %}t{% else %}T{% endif %}rue si le message imprimé correspond à l'original, en tenant compte d'éventuelles frappes longues. Ou {% if is_js %}f{% else %}F{% endif %}alse s'il y a des erreurs ou s'il n'y a pas de lettres longuement frappées.
